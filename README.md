# GoalKeeper
GoalKeeper — это мобильное приложение для управления и генерации целей на основе выделенного времени. Идея в том, чтобы помочь пользователям планировать и организовывать свои задачи, основываясь на сложности целей и количестве времени, которое пользователь готов потратить.
## Основные функции

- **Добавление целей**: Пользователи могут вручную добавлять цели с указанием их сложности.
- **Генерация целей**: Приложение автоматически генерирует список целей в зависимости от времени, которое пользователь гибко настраивает.
- **Гибкая настройка**: Приложение позволяет установить свои характеристики сложности целей; указать время, которое бы пользователь хотел потратить на свои задачи.
- **Отслеживание выполнения целей**: Пользователь может помечать цели как выполненные.
- **Фильтрация и поиск целей**: Функция поиска позволяет находить цели по ключевым словам.
- **Навигация**: Приложение использует навигацию по вкладкам, чтобы легко переключаться между основными экранами.

## Технические детали

Приложение разработано с использованием следующих технологий:

- **Язык**: Kotlin
- **Архитектура**: MVVM (Model-View-ViewModel)
- **Jetpack Compose**: для построения UI
- **Room**: для работы с локальной базой данных
- **Navigation Component**: для управления навигацией между экранами
- **Корутины**: для выполнения асинхронных операций
